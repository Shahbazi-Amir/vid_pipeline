# Financial Video RAG Pipeline

پایپ‌لاین ماژولار برای دریافت ویدئوهای آکادمی هوش مالی، استخراج و استانداردسازی صوت، تولید متن فارسی زمان‌دار، آماده‌سازی فایل بازبینی و ساخت خروجی سازگار با مخزن `Shahbazi-Amir/transcription`.

## هدف پروژه

این پروژه ابزار پردازش را نگهداری می‌کند. خروجی نهایی قسمت‌های تأییدشده در مخزن `transcription` و branch زیر قرار می‌گیرد:

```text
agent/financial-rag-transcripts
```

## فایل اصلی

```text
src/vid_pipeline/cli.py
```

برای اجرای رایگان روی Google Colab نیز یک Notebook جداگانه ساخته خواهد شد.

## مسیر کلی پردازش

```text
URL یا عبارت جست‌وجو
→ کشف و اعتبارسنجی منبع
→ دانلود ویدئو
→ استخراج صوت با ffmpeg
→ تبدیل به mono و 16kHz
→ تولید متن فارسی زمان‌دار با faster-whisper
→ ذخیره فایل خام و metadata
→ تولید نسخه قابل بازبینی
→ بازبینی انسانی با تطبیق صوت
→ ساخت Markdown نهایی
→ ساخت و اعتبارسنجی JSONL برای RAG
```

## اصول طراحی

- بخش‌های دانلود، صوت، رونویسی، بازبینی و RAG ماژول‌های جدا هستند.
- اجرای مجدد نباید مراحل تکمیل‌شده را بی‌دلیل تکرار کند.
- کد باید روی macOS، Google Colab و Ubuntu قابل اجرا باشد.
- ویدئو و صوت حجیم وارد Git نمی‌شوند.
- خروجی Whisper به‌تنهایی متن نهایی محسوب نمی‌شود.
- وضعیت `reviewed` فقط پس از بازبینی واقعی ثبت می‌شود.
- هر قسمت مستقل پردازش، اعتبارسنجی و commit می‌شود.

## الگوی خروجی مقصد

```text
raw/<collection>/episode-01.raw.json
transcripts/<collection>/episode-01.md
rag/episodes/<collection>-episode-01.jsonl
manifest.json
```

نام‌گذاری دقیق با ساختار موجود مخزن `transcription` هماهنگ نگه داشته می‌شود.
